## 🚀 Quick Start

### Local Development

1. Clone the repository:
```bash
git clone https://github.com/raone1537/kael-core-ai.git
cd kael-core-ai
```

2. Open `index.html` in your browser:
```bash
open index.html  # macOS
start index.html # Windows
firefox index.html # Linux
```

## 🔧 Installation

### GitHub Pages Setup

1. Repository is already set up for GitHub Pages
2. Visit your repo settings → Pages
3. Select `main` branch as source
4. Your site will be live at: `https://[username].github.io/kael-core-ai/`

### Custom Domain

1. Create a `CNAME` file in the root with your domain
2. Configure DNS records pointing to GitHub Pages
3. Update the domain in repo settings

## 📝 Customization

### Colors

```css

  background: radial-gradient(circle at top, #1f2937 0%, #0f172a 45%, #020617 100%);
  color: #e5e7eb;
}
```

### Content
Edit text directly in HTML sections (hero, features, pricing, etc.)

### Images
Replace or add images in the `<img>` tags

### Forms
Connect newsletter and contact forms to your backend service
